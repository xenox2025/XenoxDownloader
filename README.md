# 🤖 Xenox Downloader

Descargador de YouTube simple, rápido y seguro.

## ✨ Características

- 📹 Descarga videos en 480p
- 🎵 Extrae audio en MP3
- ⚡ 5 hilos de descarga (rápido)
- 🔋 Bajo consumo de batería y RAM
- 🔒 Open source y seguro
- 🇨🇺 Creado en Cuba por Toohuman & Xenox
